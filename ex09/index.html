Menções e Códigos

código em formato de texto

Quando escrevemos algum código de programação, sua extrutura fica sem formatação. para esse tipo de situação existe uma tag chamada <code>

Pré formatação

Quando queremos digitar algo e queremos que ele continue da forma que foi escrito, por exemplo um código fonte. Utilizamos a tag <pre>
exemplo: 

 se (20%2=0 ) entao
    escreva(" este número é par")
    senao 
    escreva("este numero é impar")

Note que o texto acima ficou com formato de um código, pois foi esta maneira que ele foi escrito.

citaçoes de um texto

Para citações de uma frase, utilizamos tag <q>
Por exemplo:
Como diria um amigo O computador é apenas um burro, mais veloz
Note que a citação ficou entre aspas.

Citaçoes em bloco

Geralmente usamos quando fazemos uma citação maior. Trecho de um texto, um livro etc..
usamos tag <blockquote> também podemos endereçar URL do site que foi sitado
