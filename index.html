Utilizando a tag adress

Esse exercício, é uma pratica de formatação de textos. como utilizar algumas tags. no html tudo tem um significado, por exemplo ao digitar um endereço a gente utiliza tag <adress> que ela automaticamente irá deixar da seguinte forma :

Rua guarani, 60 quintino Bocaiuva, RJ

Utilizando Negrito:

No itálico, utilizamos a tag<strong> para indicar uma palavra de força, por isso ela fica em negrito

utilizando tag Itálico

Geralmente é utilizado quando queremos da ênfase em uma palavra. usamos a tag <em> assim ela põe a palavra em Itálico

Texto Menor

O texto menor, as vezes é utilizado pra comentários ou observações no final da págnia do site. Nesse caso utilizamos a tag <small> e o texto ficará menor

Texto DELETADO

Quando precisamos sinalizar que palavra foi deletada, utilizamos tag <DEL>. Essa tag fará com que o texto apareça RISCADO

texto inserido

Essa tag é utilizada pra mostrar um texto que foi inserido utilizamos o <ins> e o texto irá aparecer com uma sublinha.

texto sobrescrito e subescrito

Para utilzarmos números ou letras menores, como na matemática ou fórmula quimica. usamos as tags <SUB> e a <SUP>
Exemplo: H2O tag sub.
x2 tag SUP

Marcação de texto

Outra função utilizada para destacar ou marcar uma palava é a tag <mark>. Nesse comando o texto em questão ficará Marcado igual marca texto que utilizamos quando marcamos um livro
